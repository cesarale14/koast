Kast
01 · the mark

One symbol, three forms.

The Koast mark is a banded circle representing accumulated geological strata — the visual metaphor for the AI co-host accumulating context, memory, and learning. The mark functions as the o in the wordmark "Koast", and stands alone as a symbol. Three forms cover all sizes:

5-band — primary

Use at ≥48px. Five strata, geological proportions, full gradient.

3-band — favicon

Use at <48px. Three strata, full structural integrity at small sizes.

Kast

Wordmark-only

Use at <16px or in single-color contexts where the gradient can't survive.

The size rule

A single rule governs which form to use:

contextformwhy
font-size ≥ 48pxwordmark + 5-band oBands are clearly distinguishable
16px ≤ font-size < 48pxwordmark + 3-band o5-band collapses to "circle with gradient" at this scale
font-size < 16pxwordmark-onlyEven 3-band can't hold structure; the typographic mark survives alone
single-color print / faxwordmark-onlyGradient can't reproduce in monochrome; flat-fill the mark only as last resort
02 · usage contexts

When to use what.

Wordmark + banded o (default)

Use almost everywhere. Marketing site, app header, dashboard chrome, email signatures, presentations, business cards. Anywhere the brand needs to communicate identity AND name.

Most contexts → this option.

Banded circle alone

Use only when context labels itself. Favicon, app icon, in-app Koast avatar (chat bubble), social profile photo. The surrounding context (URL bar, app drawer, profile name) tells the user this is Koast.

Standalone → only with self-labeling context.

Wordmark-only fallback

Use at sub-16px or single-color contexts. Footer fine-print, terms of service, fax-grade print, stamps, embroidery. Anywhere the banded structure cannot reproduce.

Edge cases only. Default to the banded form whenever possible.

Animated mark

Three motion registers, each with a job. Active (cascade) while AI is processing. Milestone (deposit) on positive completion events. Marketing hero (continuous cascade) on the public landing page only.

Static is the default. Animate intentionally, not constantly. Full vocabulary in Section 05.

Color usage rules

  • Tide #4cc4cc is the primary brand teal. Use for accent, hover, focus, primary CTAs, active states.
  • Trench #0e7a8a is the deep contrast variant. Use for filled buttons on light bg, active emphasis, brand-darkmark accents.
  • Shore Mist #d4eef0 is the lightest variant. Use for tinted surfaces, subtle backgrounds, hover-state fills, success states.
  • Avoid introducing teals outside this five-stop ramp. The gradient sequence is the brand's tonal signature.
04 · typography

Plus Jakarta Sans, locked.

Plus Jakarta Sans is the brand typeface across all properties. Sourced from Google Fonts (open-source, SIL Open Font License). Available weights as needed; brand voice anchors on weight 800 for the wordmark and 500 for body.

05 · motion

A layered motion vocabulary.

Three registers, each with a distinct gesture and trigger condition. Idle is static — the default. Active is the cascade — bands fade through in sequence while Koast is processing. Milestone is the deposit — a one-shot celebration on specific completion events. The motion does real work: it tells the user what Koast is doing, when something happens, and when it's at rest.

Idle — static

No motion. The default state. Used in app chrome, favicons, email signatures, anywhere the brand mark just exists.

Active — cascade

3s cycle · 130ms stagger · opacity 1→0.55. Bands fade top-down sequentially. Reads as consulting layers. Triggered while AI is processing.

Milestone — deposit

2s event, one-shot · 18px stack shift · ghost band drops in. Reads as memory just stored. Fires on positive completion events.

Trigger conditions

registerfires for
Active (cascade)AI generating reply · pricing engine recalc · booking sync in flight · channel manager re-sync · iCal poll · onboarding analysis
Milestone (deposit)Onboarding complete · first listing synced · first AI message handled · first booking received · daily pricing sync done · weekly review sent · 30-day anniversary
Silent (idle)Routine API calls · page navigation · UI interactions · errors (use different visual) · background polling without user impact

Small-size active fallback

At 16-31px, cascade's per-band opacity changes are too subtle to perceive. Use the pulse fallback: a whole-mark CSS filter (brightness 1.0 → 1.12, saturation 1.0 → 1.10) over a 1.6s ease-in-out cycle. The mark "lights up" while thinking — visible at any pixel resolution. Below 16px, omit active state entirely; favicons should not animate.

Marketing hero variant

For one specific surface — the public marketing landing hero — the cascade runs continuously without rest periods. 2.4s cycle, negative stagger delays for immediate desync. Same gesture as active, no quiet phase. Single context only: never propagated to body, pricing, blog, or any in-app surface.

Where motion does not live

Active state with motion is for live web/app contexts only. The following surfaces are locked as pure static — all bands at full opacity, evenly stacked, no motion implied: OG / social cards · app icons (iOS, Android, Windows) · all favicon sizes · print / PDF brand assets · email signatures. The motion vocabulary does its work on the actual site, not on share previews or app launchers. A first-time viewer seeing the brand on a Twitter or LinkedIn share card needs the cleanest read, not a degraded mid-fade frame.

06 · clear space + minimum sizes

Give the mark room.

Clear space rule

Maintain a clear-space margin around the mark equal to half the diameter of the banded circle (½× o-height) on all sides. No other graphic, text, or interactive element may enter this zone.

07 · forbidden modifications

Hands off the system.

do
  • Use the 5-band variant at ≥ 48px font-size.
  • Use the 3-band variant at < 48px font-size.
  • Use the wordmark-only fallback at < 16px or in single-color contexts.
  • Maintain the locked color palette across all assets.
  • Animate sparingly — cascade for active, deposit for milestone, hero-cascade only on marketing landing.
  • Respect the clear-space rule.
  • Render text in Plus Jakarta Sans, weight 800 for the wordmark.
don't
  • Don't rotate the banded o (band orientation is brand-meaningful).
  • Don't change the band count — only 5 or 3, per the size rule.
  • Don't recolor outside the cool teal palette.
  • Don't add a stroke, glow, drop shadow, or other effect to the mark.
  • Don't change letter-spacing on the wordmark.
  • Don't substitute a different typeface for the wordmark.
  • Don't use continuous downward motion (creates barber-pole register).
  • Don't use the snappy 3s animation tempo (reserved as forbidden).
  • Don't place the mark on photographic / patterned backgrounds — always solid.
  • Don't enclose the mark in a box, badge, or container shape.
